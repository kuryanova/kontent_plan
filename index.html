Синхронизация

Сейчас данные хранятся в localStorage браузера и доступны только на этом устройстве. Чтобы синхронизировать между устройствами, подключи Supabase.

Как настроить таблицу в Supabase +

1. Создай проект на supabase.com

2. В разделе SQL Editor выполни:

create table content_studio (id text primary key default 'default', data jsonb, updated_at timestamptz default now());

3. В Authentication отключи RLS на таблице или настрой политики

4. Скопируй URL и anon-ключ из Settings → API
